# AI customer support chatbot

## Project Description

This project is an AI-powered customer support chatbot built using React.js and FastAPI.  
The chatbot is designed to automate customer interaction using Natural Language Processing (NLP) and Machine Learning techniques.

This project is being developed as part of the NextGen Forge Technologies AI/ML Internship Program.

--

## WEEK 1 :

### Tech Stack installed:

- React
- Vite
- FastApi
- NLTK
- Uvicorn

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
